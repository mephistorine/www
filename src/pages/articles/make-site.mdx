Выписывание SSL сертификатов

```bash
sudo certbot certonly --standalone -d mephi.dev -d www.mephi.dev
```

Сборка докер образа

```bash
docker build -t mephi .
```

Запуск

```bash
docker run -d -p 80:80 -p 443:443 -t mephi
```
